## 🛠️ Used Technologies/Tools
* Expo Router
* React Native
* NativeWind (TailwindCSS in React Native)
* React Native Gesture Handler
* Reanimated
* Typescript

## How to Clone the Project?

### First Step
``` terminal
  git clone https://github.com/Gabrielst03/gmail-ui-clone.git
```

### Second Step
``` terminal
  cd gmail-ui-clone
```

### Third Step
``` terminal
  npm install
```

### Fourth Step
``` terminal
  npx expo start
```
